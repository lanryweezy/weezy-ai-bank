# Weezy AI Core Banking System (CBS) - Project Manifest

## 🚀 Overview
Weezy AI is a fully agentic, AI-orchestrated Core Banking System (CBS) localized for the Nigerian market. It combines high-performance double-entry accounting with real-time AI security and an industrial-grade "Luxury Banking" interface.

## 🛠️ Tech Stack
- **Backend**: Python 3.9+ (FastAPI)
- **Database**: PostgreSQL (SQLAlchemy with composite indices and row-level locking)
- **AI Core**: Google Gemini 1.5 Pro (Orchestration & Reasoning), Gemini 1.5 Flash (Fraud Screening)
- **Frontend**: React 18 (Vite, TypeScript, Tailwind CSS, Shadcn/UI)
- **Design System**: Glassmorphism with High-Contrast Fintech Black typography.

## 📦 System Architecture (22 Specialized Modules)

### Core Ledger & Transactions
1.  **Accounts & Ledger Management**: Atomic, double-entry core with hierarchical GL support.
2.  **Transaction Management**: High-velocity switch with NIP (NIBSS) and internal transfer hooks.
3.  **Financial Switch**: Real-time monitoring of ISO-8583 message streams.

### AI & Intelligence
4.  **Cognitive Core (Prime)**: Central AI orchestrator with native function calling.
5.  **Intelligence Terminal**: Natural language data CLI translating questions to safe SQL.
6.  **AI Fraud Shield**: Real-time transaction screening and autonomous blocking.
7.  **AI Credit Analyst**: Automated loan appraisal using DTI and Nigerian economic context.

### Retail Banking
8.  **Customer Identity (CIM)**: Tiered KYC management (Tier 1/2/3) with NIN/BVN verification.
9.  **Biometric Vault**: Gemini Vision-powered facial match for Tier 3 onboarding.
10. **Card Center**: Verve & Mastercard lifecycle management (Virtual/Physical).
11. **Mobile Wallet**: P2P settlement hub for customers.
12. **Fixed Vault**: High-yield term deposits with automated daily accrual.
13. **Lifestyle Hub**: Instant utility, airtime, and data provisioning core.

### Business & Field Operations
14. **Agent Banking (SANEF)**: Distributed node network management.
15. **Revenue Hub**: Automated agent commissions & revenue split engine.
16. **Merchant Ops**: POS terminal management & T+1 settlement logic.
17. **Corporate Payroll**: Bulk disbursement engine with AI anomaly auditing.
18. **Virtual Accounts**: Collection-only NUBAN infrastructure for merchants.

### Governance & Compliance
19. **Dual Control (Maker-Checker)**: Centralized authorization for high-risk operations.
20. **Regulatory Core**: Automated CBN FinA, CRMS, and NFIU CTR export engine.
21. **System Heartbeat (EOD)**: Unified batch lifecycle (Interest, Recurring, Reminders).
22. **Alerts Hub**: Real-time SMS/Email notification engine with automated fee collection.

## 🇳🇬 Nigerian Localization Standards
- **NUBAN**: Official CBN checksum algorithm implemented.
- **NIP/NQR**: Standardized protocols for instant payments and QR.
- **SANEF**: Agency banking standards for location and tiering.
- **Taxes**: Automated ₦50 Stamp Duty & 7.5% VAT multi-leg ledger postings.
